# CSV Reports Generator

Утилита для анализа и генерации отчетов из CSV файлов с данными сотрудников. Программа обрабатывает информацию о производительности, задачах и навыках команды, предоставляя структурированные аналитические отчеты.

Для добавления нового отчета использовать базовый класс BaseReport как родительский.

## Быстрый старт

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск отчета по эффективности
python main.py --files data1.csv data2.csv --report performance

# Запуск тестов
pytest tests/ -v
